🌍 AI Travel Recommendation System

An interactive AI-powered travel recommendation web app built using Streamlit and Machine Learning (KNN algorithm).
This app suggests travel destinations based on user preferences like budget and travel category.

🚀 Features
🎯 Personalized travel recommendations
💰 Budget-based filtering (Low / Medium / High)
🌄 Category selection (Beach / Hill / City)
🖼️ Beautiful destination images
⭐ Attractions for each place
📍 Nearby places suggestions
🏨 Hotel recommendations
🌍 Direct Google Maps link
🎨 Modern glassmorphism UI design
